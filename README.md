# ☁️ Weather Dashboard


![SharedScreenshot](https://github.com/user-attachments/assets/da1892be-3df6-41cf-aac7-f7a64b0e0d43)

A beautiful, responsive weather dashboard built with vanilla JavaScript and the OpenWeatherMap API. Features real-time weather data, 5-day forecasts, and location search.

## ✨ Features
- Current weather conditions (temperature, humidity, wind, pressure, UV index)
- 5-day weather forecast
- Search by city name
- Recent cities (saved locally)
- Toggle between °C and °F
- Weather-appropriate tips
- Fully responsive design
- Beautiful weather icons

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API
- Font Awesome Icons


## 📦 Installation
1. Clone the repository
2. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
3. Add your API key to `script.js`
4. Open `index.html` in your browser

## 📝 Usage
1. Enter a city name in the search box
2. Press Enter or click Search
3. View current weather and 5-day forecast
4. Click on recent cities to quickly switch locations
